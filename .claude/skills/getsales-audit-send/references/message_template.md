# LinkedIn Message Template — Audit Delivery

## Default template (direct ecom brands)

```
Hey {first_name},

Ran a comparison of how AI agents recommend {category}. {company_name} vs. a few competitors.

Full breakdown here: {audit_link}

A few of the gaps there are actually quick to close. Happy to walk you through it if useful.
```

## Marketplace variant

For companies that ARE marketplaces (not brands selling products):

```
Hey {first_name},

Ran a comparison of how AI agents surface online marketplaces when shoppers search for products in {market}. {company_name} vs. the usual suspects.

Full breakdown here: {audit_link}

A few of the gaps there are actually quick to close. Happy to walk you through it if useful.
```

## B2B/industrial variant

For B2B or industrial companies:

```
Hey {first_name},

Ran a comparison of how AI agents recommend suppliers in the {category} space. {company_name} vs. a few competitors.

Full breakdown here: {audit_link}

A few of the gaps there are actually quick to close. Happy to walk you through it if useful.
```

## Category phrasing guidelines

- Strip prompt boilerplate: "best online stores for X" → just "X"
- Make it sound like a conversation: "sports nutrition brands" not "sports nutrition supplement products"
- Translate non-English categories to English
- Keep it short: 2-4 words ideal
- Use "brands" for consumer products, "suppliers" for B2B, "retailers" for multi-brand stores
